Zweidimensionale Datenbindung

Controller- und Model-Code in externen JavaScript-Datei: HauptController.JS

function AdressbuchController ($scope) {	
	$scope.benutzer = [
	{"vorname": "Max", "name": "Meier"},
	{"vorname": "Florian", "name": "Wiegand"},
         ...
         ...	
	];	
}

Controller besitzt immer eine $scope-Variable

$scope-Objektvariable für Verbindung zum View

Zweidimensionale Datenbindung

Body des HTML-Dokuments (View)

<body ng-controller=“AdressbuchController">
 
    <label >Vor- oder Nachnamen eingeben:</label>
    <input type="text" ng-model="suche" placeholder="Suchfeld">


    <ul>
  		<li ng-repeat=“person in benutzer |  filter:suche">
                         Nachname: { { person.name } } <br /> 
                         Vorname: { { person.vorname } } <br /> 
			 Weitere Daten: <br /> <br /> 
                </li>
    </ul>

</body>

ng-repeat iteriert über jedes Objekt aus dem Array benutzer

Dependency Injection

Objekte sind wiederum von anderen Objekten abhängig

Möglichkeiten zur Instanziierung von Objekten und Definition von Abhängigkeiten

  1. Innerhalb einer Funktion über new-Parameter: Beispiel Adressbuch-Objekt
    2. function sucheAdressbuch(benutzer){
	var db = new Database(...);
	db.suche(benutzer);
}
  • Außerhalb einer Funktion: Objekt als Übergabeparameter
    • function sucheAdressbuch(db,benutzer){
	db.suche(benutzer);
}
