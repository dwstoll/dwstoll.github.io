Definition - Was ist AngularJS?

  • Sehr populäres Open Source JavaScript-Framework (komplex) Clientseitiges Applikationsframework. IN AngularJS werden HTML und JS nicht wie bei vielen anderen Frameworks weggekapselt, sndern um Funktionalitäten erweitert

  • Ziel: HTML soll fitter für Web Applikationen werden (dynamischer)

  • Von Google Inc. (Erscheinungsjahr 2009) unter der MIT-Lizenz zur kostenlosen Nutzung bereitgestellt, aktuelle Version 1.2.16 April 2014, Programmiersprache JavaScript

  • Unterstützung der Softwareentwicklung von dynamischen Webanwendungen (besonders SPA), da HTML bei statischen Dingen gut, aber dynamisch doof

  • Nach dem MVVM- bzw. MVC-Modell / Architektur

Vorteile

Vereinfachung der Softwareentwicklung von dynamischen Webanwendungen

  • Quellcode: starke Reduzierung des Umfangs, gute Lesbarkeit, verständlich (gut für Zusammenarbeit Designer und Entwickler)

  • Qualität: leichte Testbarkeit und einfaches Refactoring

  • Wiederverwendbarkeit

  • Einbindung fremder Bibliotheken möglich

  • Populäres Framework -> starke Community

Nachteile

  • Performanceverlust in manchen Fällen http://www.senaeh.de/angularjs/

  • Einarbeitung nötig

Fazit

Einfach über die AngularJS Website oder über GitHub das ganze downloaded und dann kann es schon beginnen. da angular nur den mit ng-app markierten Bereich verarbeitet, lässt es sich gut mit anderen Bibliotheken verbinden und auch in ältere Webanwendungen einbauen.
